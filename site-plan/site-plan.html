Individual Project Site Plan

Author: Zelalem Eshetu

1. Site Name

Global Study Hub

This name was selected because the website helps students discover study tips, resources, and tools to improve learning. It is simple, memorable, and clearly shows the purpose.

Optional domain: globalstudyhub.org

2. Site Purpose

The purpose of this site is to help students learn effectively by providing study guides, productivity tools, recommended resources, and motivation. The site will include simple tutorials, organized learning strategies, and downloadable materials.

3. Scenarios

  • What study techniques can help me stay focused and remember concepts better?
  • Where can I find recommended tools and resources for improving my online learning?

4. Color Scheme

Primary Color: #0b62a4 (Used for headings and accents)

Secondary Color: #f7faff (Used for background)

Accent Color: #1a1a1a (Used for text)

5. Typography

Main Font: Arial

  • Headings: Arial Bold
  • Body Text: Arial Regular
